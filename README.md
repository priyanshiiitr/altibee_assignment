## Overview

Altibee Assignment is developed as part of an assessment for Altibee. The project leverages TypeScript and CSS to provide a scalable, maintainable, and visually appealing application. The main goal is to demonstrate proficiency in application development and best practices.

## Features

- Component-based UI architecture
- Type-safe logic using TypeScript
- Responsive design using CSS
- Modular and maintainable codebase

## Tech Stack

- **TypeScript** (95.2%)
- **CSS** (4.3%)
- Other technologies (0.5%)

## Getting Started

1. **Clone the repository**
    ```bash
    git clone https://github.com/priyanshiiitr/altibee_assignment.git
    cd altibee_assignment
    ```

2. **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3. **Run the development server**
    ```bash
    npm start
    # or
    yarn start
    ```
   The app will be available at [http://localhost:3000](http://localhost:3000).

## Scripts

- `start` - Run the development server
- `build` - Build the project for production
- `test` - Run unit tests
